# Jairo Ponce - Portfolio Website

## ✨ Features
- **Responsive Design** — optimized for desktop and mobile.
- **About Section** — brief introduction and background.
- **Resume Section** — education, work experience, and expertise.
- **Projects Section** — showcase of industrial automation, web dashboards, and embedded system projects.
- **Blog Section** — placeholder for articles and updates.
- **Contact Section** — contact form + embedded Google Maps.
- **Sidebar with Social Links** — LinkedIn, GitHub, Instagram.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

### 2. Open in Browser
Simply open `index.html` in your browser.  
Or use VS Code Live Server:
```bash
# If you have Live Server extension installed
Right Click → "Open with Live Server"
```

---

## 📌 Deployment
- **GitHub Pages:** push to `main` branch and enable Pages in repo settings.  
- **Netlify / Vercel:** drag & drop the folder or connect your GitHub repo.
